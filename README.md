## How to add to KiCAD:

These instructions have been verified for KiCAD 9.0.7. Your mileage may vary.

1. Open **KiCAD**
1. Go to **Preferences â†’ Manage Symbol Libraries**
2. Select the **Global Libraries** tab
3. Click **Add existing library to table** (folder icon)
4. Browse to the `ME218_Trutna.kicad_sym` file and select it
6. Click **OK**
